#AgriAssist AI – Crop Advisory Chatbot

An AI-powered crop advisory web application designed to help farmers and field supervisors obtain agricultural guidance through natural-language conversations. The system provides recommendations related to crop diseases, pest management, irrigation practices, soil health, and post-harvest handling using a Large Language Model (LLM).

#Features

- AI-powered agricultural chatbot
- Crop disease and pest management guidance
- Farming best-practice recommendations
- User-friendly chat interface
- Mobile-responsive design
- Secure user authentication
- Chat history management

#Project Objective

The objective of this project is to provide accessible agricultural advisory support to farmers through an AI-powered assistant. The application aims to improve decision-making, reduce information gaps, and demonstrate the practical use of Artificial Intelligence in the agriculture sector.
